## 脚本式

- 系统命令和文件

```shell
python script.py
```

- Unix 风格可执行脚本：#!

```shell
# 首行添加 #!
#!/usr/local/bin/python
#!/usr/bin/env python

# 赋予执行权限
chmod +x script.py

# 运行脚本
./script.py
```
